# RU
# Алгоритмы и абстрактные структуры данных

## Описание
Реализация набора алгоритмов и задач на C++ с использованием
абстрактных структур данных.

## Реализованные задачи
- Task1 — вычисление значения арифметического выражения с использованием стека
- Task2 — работа с односвязным списком (добавление, удаление, поиск, сортировка)
- Task3 — поиск всех подмножеств с заданной суммой (алгоритм с возвратом)
- Task4 — вывод всех различных подмассивов массива
- Task5 — построение сбалансированного бинарного дерева из отсортированного массива
- Task6 — проверка изоморфности строк с использованием хеш-таблицы

## Структура файлов
├── Task1.h        # Вычисление арифметических выражений (заголовок)  
├── Task1.cpp      # Вычисление арифметических выражений (реализация)  
├── Task2.h        # Односвязный список (заголовок)  
├── Task2.cpp      # Односвязный список (реализация)  
├── Task3.cpp      # Поиск подмножеств с заданной суммой  
├── Task4.h        # Подмассивы массива (заголовок)  
├── Task4.cpp      # Подмассивы массива (реализация)  
├── Task5.cpp      # Сбалансированное бинарное дерево  
├── Task6.h        # Проверка изоморфности строк (заголовок)  
└── Task6.cpp      # Проверка изоморфности строк (реализация)  

## Язык программирования
C++

## Запуск
Открыть терминал в VS Code и выполнить (X — номер задания):
g++ TaskX.cpp -o TaskX
./TaskX

# EN
# Algorithms and Abstract Data Structures

## Description
Implementation of a set of algorithms and tasks in C++ using
abstract data structures.

## Implemented Tasks
- Task1 — evaluation of arithmetic expressions using a stack
- Task2 — singly linked list operations (push, delete, search, sort)
- Task3 — finding all subsets with a given sum (backtracking algorithm)
- Task4 — output of all distinct subarrays of an array
- Task5 — building a balanced binary tree from a sorted array
- Task6 — checking string isomorphism using a hash table

## File Structure
├── Task1.h        # Arithmetic expression evaluation (header)  
├── Task1.cpp      # Arithmetic expression evaluation (implementation)  
├── Task2.h        # Singly linked list (header)  
├── Task2.cpp      # Singly linked list (implementation)  
├── Task3.cpp      # Subset sum search  
├── Task4.h        # Array subarrays (header)  
├── Task4.cpp      # Array subarrays (implementation)  
├── Task5.cpp      # Balanced binary tree  
├── Task6.h        # String isomorphism check (header)  
└── Task6.cpp      # String isomorphism check (implementation)  

## Language
C++

## How to Run

g++ TaskX.cpp -o TaskX
./TaskX
